# 🎮 Call of Duty 4 CFG Checker

A powerful and efficient tool designed to analyze and validate Call of Duty 4 (Modern Warfare) configuration (CFG) files. This project helps players, server admins, and competitive gamers ensure their configs are clean, optimized, and compliant with expected standards.

## 🚀 Features

- 🔍 **Detects invalid or unsupported commands** - Identifies commands that won't work or may cause issues
- ⚡ **Identifies performance-impacting settings** - Highlights settings that affect game performance
- 🔒 **Flags suspicious or restricted configurations** - Spots potentially unfair or banned commands
- 📊 **Provides clear and readable analysis output** - Easy-to-understand reports
- 🛠 **Easy-to-use and lightweight** - Simple interface with minimal resource usage
- 🔄 **Supports quick and efficient CFG scanning** - Fast analysis for multiple files

## 🎯 Purpose

CFG files play a critical role in gameplay performance and fairness. This tool ensures that configuration files are:

- ✅ Clean and optimized
- ✅ Free from suspicious or restricted commands  
- ✅ Ready for competitive or server use

## 💡 Use Cases

- **Competitive players** - Verify configs before matches or tournaments
- **Server admins** - Enforce fair play and server compliance
- **Developers** - Analyze CFG behavior and debug configuration issues
